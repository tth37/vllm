# Archive Layout

This archive keeps the superseded exp1 materials that are still useful for historical debugging.

- `reproduced_qwen3_0p6b/`: the March 17 branch-based reruns of `baseline`, `dockerbe_async`, `dockerbe_shm`, and `dockerbe_full_shm`
- `root_cause_async_output/`: the profiling and A/B data used to show that async output handling was the dominant missing optimization
- `v1_results/` and `profile_results/`: the original older benchmark and profiling artifacts
- `branch_reproduction/`: the old branch-based reproduction harness
- `legacy_scripts/`: superseded benchmark entrypoints
